How does web hosting work?

When you develop a website, unless you publish it on the internet nobody can access your website. A web hosting provider rents out server space to such website owners for hosting their website on the internet. Once it's done, any user on the internet can access your website by typing your web URL in their web browser. The backend process that happens is that the user's computer connects to the server where your website is hosted and then the server responds to this request by displaying the website on the visitor's browser
